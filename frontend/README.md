## Overview

Frontend application for the Project 3 web app. Serves UI, communicates with backend API, and provides local development tooling.

## Prerequisites

- Node.js (LTS recommended, >= 16)
- npm or yarn

## Quick start

1. Install dependencies
   - npm: `npm install`
   - yarn: `yarn`
2. Start development server
   - npm: `npm run dev` or `npm start`
   - yarn: `yarn dev` or `yarn start`

## Linting

- Run the linter:
  - npm: `npm run lint`
  - yarn: `yarn lint`

## Environment variables

- If using Vite: prefix variables with `VITE_` (e.g. `VITE_API_URL`)
